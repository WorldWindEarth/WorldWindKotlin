Video on Terrain (KLV)

Drapes a real STANAG 4609 drone clip onto the terrain via the OES external-texture path (MediaPlayer → SurfaceTexture → GL_TEXTURE_EXTERNAL_OES) and animates the projected footprint with the embedded MISB ST 0601 KLV telemetry.

The video and a pre-extracted KLV timeline JSON are bundled cross-platform via moko-resources at commonMain/moko-resources/assets/video/ — accessed on Android as MR.assets.video.drone_motion_mp4 / drone_motion_json, on JVM via the bundled JAR classpath, and on JS as webpack-rewritten asset URLs. The standalone extractor at tools/ExtractKlv.java demuxes the MPEG-TS, parses ST 0601, and emits a flat {tMs, tl, tr, br, bl} JSON per telemetry sample.

Pipeline

  • MediaSurfaceTexture wraps a SurfaceTexture bound to a GL_TEXTURE_EXTERNAL_OES name allocated lazily by the engine.
  • ProjectedMediaSurface.texture takes the OES texture directly; the OES variant of the homography surface-quad shader is selected automatically.
  • Each render frame, VideoOnTerrainTutorial.tick() resamples the timeline at the player's current position and updates the surface's four corners.
