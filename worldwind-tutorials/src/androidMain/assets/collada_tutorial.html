COLLADA Tutorial

Demonstrates how to load and display a COLLADA 3D model on the globe.

This example loads the classic duck COLLADA model, places it at an absolute altitude, and supports ray-picking to display intersection points with leader lines.

Example

ColladaFragment.kt

The ColladaFragment class extends the BasicGlobeFragment and overrides the createWorldWindow method. Here we parse the DAE file asynchronously, set up an image source factory for textures, and install a tap gesture detector that performs ray intersection picking on the model.
